What is a living style guide

A style guide is a reference for people to understand how something should look and feel in relation to a project or brand (Weber, 2011). A living style guide is an evolving or dynamic guide for design. One place for all team members to reference, discuss changes and iterations of a design (Robertson, 2014).

Why is a living style guide useful to developers and designers

Static style guides have confusion about latest version, are hard to edit and keep up to date (Hagenburger, 2012). Conversely, living style guides are constructed with code allowing for easy editing and version control(GIT) (Hagenburger, 2012).

My list of living style guide tips

  • Start with site basics, grid, styles for headers and text and the colour palette (Robertson, 2014)
  • If a pattern is used more than three times add to the style guide as a pattern (Hildebrandt, 2015)
  • Use a tool like Fabricator to help organise and generate a style guide form toolkit code (Haritonov, 2015)
  • Use KSS, Knyle Style Sheets to help keep CSS human readable yet machine parsable and easy to remember (Robertson, 2014)
  • Use SASS: Syntactically Awesome Style Sheets. This CSS addon gives you the ability to use varibles in CSS code (Haritonov, 2015)
